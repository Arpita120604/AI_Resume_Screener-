# AI Resume Screening System

An AI-powered web application that screens and ranks resumes based on job descriptions using NLP techniques.

## Features
- Upload multiple resumes (PDF/DOCX)
- Paste job description
- AI-based resume ranking
- Match score generation

## Technologies Used
- Python
- NLP
- TF-IDF
- Cosine Similarity
- Streamlit

## How it Works
1. Extracts resume text
2. Cleans and processes text using NLP
3. Converts text to TF-IDF vectors
4. Compares resumes with job description
5. Ranks candidates based on similarity
